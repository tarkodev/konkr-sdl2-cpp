Detailed Description

RAII wrapper around SDL_Texture.

Manages creation, rendering, color modulation, and cleanup of an SDL_Texture. Implements BlitTarget for compositing textures and enable_shared_from_this to safely hand out std::shared_ptr<Texture>.
