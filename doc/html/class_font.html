Detailed Description

Wraps SDL_ttf TTF_Font similar to pygame.font.Font.

Usage example: Font font(renderer, "./assets/fonts/arial.ttf", 24); Texture textTexture = font.render("Hello, SDL!", {255,255,255,255}); textTexture.blit(...);
