inline

Verify the return status of an SDL call.

Throws a std::runtime_error if the status indicates failure (non-zero), appending the SDL error message from SDL_GetError().

Known SDL2 bugs can produce spurious errors; the commented-out block below shows how to log non-critical messages instead of throwing.

Parameters
statusReturn value from an SDL function (0 on success).
errmsgDescription of the operation being checked.
