Detailed Description

Manages system cursors (mouse pointers).

Provides an interface to initialize, switch, and update the cursor based on UI interactions (hover, click, text entry, etc.).

◆ SystemCursor

enum class Cursor::SystemCursor
strong

Supported system cursor types.

  • Any: No preference; leave current.
  • Arrow: Standard arrow pointer.
  • Text: Text-input I-beam cursor.
  • Hand: Hand pointer for clickable elements.
