Detailed Description

Logical element placed on a Cell (unit, building, neutral hazard, etc.).

Inherits from Displayer, so it has a position and rendering capabilities. Provides pure virtual methods for strength, cost, upkeep, and a display function. Tracks a 'lost' state to indicate destruction or removal from the game.
