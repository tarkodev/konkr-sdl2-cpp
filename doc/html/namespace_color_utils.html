◆ textInBlack()

const bool ColorUtils::textInBlack ( const SDL_Color & c)
inline

Determines whether black text is appropriate over the given background color.

Calculates luminance using 0.2125*(r/255*100) + 0.7154*(g/255*100) + 0.0721*(b/255*100)

Parameters
cBackground color.
Returns
true if luminance >= 30, suggesting black text; false otherwise.
